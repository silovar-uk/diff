# Text Review Studio v0.3.0

修正前後の比較、表記統一、掲載用HTML整形を、**外部APIへ本文を送らずに**ブラウザ内で行う静的Webアプリです。

## できること

- 修正前／修正後の差分レビューと「確認済み」管理
- ルールベースの表記統一候補の採用・除外
- URL、メールアドレス、既存HTMLタグを保護した置換
- 行頭の `【ラベル】` を `span.info24-label` に変換
- 表記統一後テキスト・HTML・差分レポートのコピー
- 最終確認用4カラム監査ビュー
- 作業データのJSON書き出し／読み込み
- 明示的にONにした場合のみ、この端末のブラウザ領域へ自動保存

## ファイル構成

```text
assets/app-icon.png       アプリアイコン
index.html                アプリ本体

app.js                    ローカル処理ロジック

```

## GitHub Pagesで公開する

1. このZIPを解凍し、**中身のファイルをそのまま**GitHubリポジトリ直下へ配置します。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Build and deployment` で `Deploy from a branch` を選択します。
4. Branchを `main`、Folderを `/ (root)` に設定して保存します。
5. 数分後、表示されたURLで公開されます。

## セキュリティと保存

- 本文を外部APIへ送信する実装はありません。
- 外部フォント、解析タグ、CDNは使用していません。
- 初期設定では、ブラウザのローカル保存もOFFです。
- 「設定」から自動保存をONにした場合のみ、作業内容を同じブラウザのローカル領域へ保存します。
- 共有PCでは自動保存をOFFにし、必要に応じて「作業データを書き出す」を使用してください。

## 注意

- HTML構造プレビューは、実際のCMSや本番サイトのCSS・制約を完全再現するものではありません。
- 表記統一は候補提示を主目的としています。固有名詞、引用文、契約・法務表現などは必ず人が確認してください。
